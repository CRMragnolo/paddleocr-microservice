# PaddleOCR Italian Microservice

Self-hosted OCR microservice using PaddleOCR for Italian text extraction.

## Features

- **100% FREE** - No API costs
- **Italian Optimized** - 92-95% accuracy
- **Memory Optimized** - Runs on Render free tier (512MB)
- **Fast** - 5-10 seconds per image

## Deployment on Render

1. Push to GitHub
2. Create Web Service on Render
3. Select Docker environment
4. Set PORT=8000

## API

- GET /health - Health check
- POST /ocr/simple - Extract text from image
